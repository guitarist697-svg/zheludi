# Split Party OCR

Первый React-прототип приложения для деления чеков.

## Что уже есть

- Участники
- Загрузка фото чека
- OCR через Tesseract.js прямо в браузере
- Автоматическая попытка распарсить позиции и суммы
- Выбор, кто что ел
- Расчет:
  - прямые долги
  - минимум переводов

## Как запустить

```bash
npm install
npm run dev
```

После запуска открой ссылку из терминала.

## Важно

OCR в браузере — это MVP. Для хорошего коммерческого качества лучше потом добавить серверный OCR:
- Google Cloud Vision
- AWS Textract
- Mindee
- OpenAI Vision API
